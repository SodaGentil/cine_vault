## 顺序表
### 定义和初始化

#### 静态分配
```
//定义
#define MaxSize 50
typedef struct {

 Elemtype data[MaxSize];
 int length;              //线性表当前长度
}SqList;
//初始化
void InitList(SqList &L){

 L.length=0;
}
```
#### 动态分配
```
//定义
#define InitSize 100
typedef struct{

 Elemtype *data;
 int MaxSize,length;
}SeqList;
//初始化
void InitList(SeqList &L){

 L.data=(ElemType *)malloc(InitSize*sizeof(ElemType));
 L.length=0;
 L.MaxSize=InitSize;
}
```
